## 关键发现

1. **adb_ui 接口契约**:
   - `ui(keyword=None, raw=False) → list[dict]`
   - `tap(x, y) → None (adb shell input tap)`
   - dump 路径: `_dump_u2()` (uiautomator2) → `_dump_native()` (subprocess adb) → `_parse_xml()`
